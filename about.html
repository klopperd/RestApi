My REST-API


To access your website use : https://stark-citadel-41801.herokuapp.com Where stark-citadel-41801 is replaced with your app name

This is to directly open your web app

$ heroku open

This is to push latest version to Heroku

$ git push heroku master

This is to see your full log

$ heroku logs --tail

This is to view only app logs

$ heroku logs --source app --tail

This is to build your app for use on Heroku (On Heroku it needs the app name without the .exe

$ go build -o bin/restapi -v .

This is for local use

$ go build -o bin/restapi.exe -v .

This is to get your database info

$ heroku config:get DATABASE_URL

Very important in go program

To run on Heroku (they don't allow setting port) : port := os.Getenv("PORT")

To run locally you need to specify port : port := "3000"
